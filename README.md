# MessageHub App

This is a Python-based desktop messaging application.

### Features:
- Send email using Gmail
- Future plan: SMS sending via API
- Built with tkinter, smtplib, and email.mime
- Converted to .exe using pyinstaller

### Run the app:
```bash
python app.py
